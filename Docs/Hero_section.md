# Hero Section (Header + Hero) Specification

## Overall Layout

The hero occupies almost the entire first viewport (approximately 90–100vh) and consists of a premium floating header followed by a two-column hero section. The overall design is clean, spacious, luxurious, and centered inside a large rounded white container placed over a very light warm-gray background.

---

# Header

## Position

- Floating at the top with generous spacing from the page edges.
- Centered horizontally.
- Does not touch the browser edges.
- Sits inside the same rounded white container as the hero.

---

## Layout

```
------------------------------------------------------------
Logo          Navigation                 CTA Button
------------------------------------------------------------
```

### Left

Restaurant Logo

Contains:

- Gold minimalist fork emblem
- Restaurant name
- Small subtitle below

```
SAVOR
FINE DINING
```

---

### Center

Navigation menu inside a floating pill-shaped container.

Menu Items

- Home (Active)
- Menu
- About
- Reservations
- Contact

Characteristics

- Horizontally centered
- Equal spacing
- Rounded capsule background
- White background
- Soft shadow
- Gold underline beneath active item
- Active text colored gold
- Other items dark gray

---

### Right

Primary CTA Button

```
Reserve a Table
```

Characteristics

- Gold background
- White text
- Pill shape
- Medium size
- Soft shadow
- Slight hover elevation

---

# Hero Section

Immediately below the navigation.

Uses a two-column layout.

```
---------------------------------------------------------
|                                                       |
| Left Content            Right Hero Image              |
|                                                       |
---------------------------------------------------------
```

Width Distribution

- Left: ~45%
- Right: ~55%

Large horizontal spacing between both sides.

---

# Left Column

Contains all textual content.

Order

1. Main Heading
2. Supporting paragraph
3. CTA Buttons

---

## Heading

Large bold typography.

Approximately 4 lines.

```
Experience Fine
Dining Like
Never Before
```

Characteristics

- Pure black
- Extremely bold
- Large font size (60–72px)
- Tight line-height
- Left aligned
- Highest visual priority after the food image

---

## Description

Placed below heading.

Two short lines.

Muted gray color.

Medium line-height.

Small font.

Purpose

Briefly explains the restaurant experience without competing with the heading.

---

## CTA Buttons

Two buttons placed horizontally.

### Primary

```
Reserve a Table
```

Style

- Gold background
- White text
- Fully rounded
- Medium shadow

---

### Secondary

```
View Menu
```

Style

- White background
- Thin gold border
- Dark text
- Rounded pill
- Minimal shadow

Spacing between buttons is generous.

---

# Right Column

Contains a massive premium food photograph.

The image dominates the hero.

---

## Main Image

Large white ceramic plate.

Contains

- Thick grilled steak
- Fresh asparagus
- Small herb garnish

Characteristics

- Professional food photography
- Soft natural lighting
- Extremely high resolution
- Plate viewed from slightly elevated angle
- Large enough to extend toward hero boundaries
- Very soft shadow beneath plate

---

# Floating Rating Cards

Position

Contents


- Rounded capsule
- Soft shadow

- Dark rating text

Creates depth and social proof.

# Floating Decorative Ingredients

Small food ingredients scattered around the hero.

Includes

- Tomato slice

Placement

- Tomato near upper center
- Mushroom near lower center

Characteristics

Purpose

---

# Background

Predominantly white.

Subtle warm radial gradient behind the plate.

Very soft cream/golden glow.

No visible patterns.

Creates focus around the food image.

---

# Container Styling

Entire hero sits inside a premium white card.

Characteristics

- Large border radius (28–36px)
- White background
- Very soft shadow
- Large internal padding
- Plenty of whitespace
- Floating effect against dark outer background
